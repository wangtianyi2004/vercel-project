Hook 触发
01
getOrCreateContext()
Context
获取或创建 TraceContext — 通过 channelId / runId 查找已有上下文，不存在则新建并注册到内存 Map
查找 contextByChannelId 查找 contextByRunId 返回 TraceContext
02
createSpan()
Span
构建 SpanData 对象 — 填充 attributes、input/output，通过 parentSpanId 建立父子关系
name span 名称 type entry|model|tool|step attributes 键值对 构建 SpanData
03
LangfuseExporter
Export
调用 OpenTelemetry Tracer API — 将 SpanData 转换为 OTel Span 并提交至处理管线
.export() 导出 span .startSpan() 开启 span .endSpanById() 结束 span openSpans Map 管理
04
BatchSpanProcessor
Batch
批量处理队列 — 累积 Span 后统一发送，减少网络请求频次
maxExportBatchSize 10 spans scheduledDelay 5000ms BasicTracerProvider 驱动
05
OTLPTraceExporter
HTTP
序列化为 Protobuf 格式，通过 HTTP POST 发送至 Langfuse 端点
POST /v1/traces 格式 Protobuf 认证 Authorization Header → Langfuse Cloud
✓
